
## Requirements

* (Mini)conda
* Python 3.10+
  * Python 3.8+ may work.

## Install

Setup python environment using conda.

```
$ conda create -n jp-llama-experiment python=3.10
$ conda activate jp-llama-experiment
```

Install modules.

```
$ python -m pip install -r requirements.txt
```
